About Hamro Notes

Hamro Notes is an online notes creating website that create notes and save it into the local storage of your browser. You can create, delete, edit and search for your notes. This website is still under developnment and requrires some more improvements like the feature of marking note as important, sorting the note etc.

Hamro Notes is a simple and easy-to-use website that helps you keep track of your notes and ideas. You can create new notes, edit and manipulate them, save them for later, and delete them when you're done.
